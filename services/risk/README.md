# Phase 7 approval and risk boundary

This package owns immutable, fail-closed approval assessments for synthetic simulated-paper
research. It consumes exact Phase 6 lineage, separately versioned policy and scope evidence,
human-controlled synthetic authorization evidence, revocation events, and server-owned risk-input
fixtures. Every assessment remains governance-only: it grants no execution authority, provides no
investment advice, and makes no real-performance claim.

There is no broker, submission, fill, position, paper-execution, or live-capability implementation in
this package. Risk inputs are synthetic exposure scenarios used only to exercise policy checks; they
are not executable intents.
